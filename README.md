# Packeo — Site vitrine

Site one-page premium pour Packeo — emballage au prix manufacturier direct.

Construit avec **Vite + React + Tailwind CSS**.

## Développement local

```bash
npm install
npm run dev
```

Site accessible sur http://localhost:5173/

## Build production

```bash
npm run build
npm run preview
```

## Déploiement

Le site est déployé automatiquement sur **GitHub Pages** via GitHub Actions
(`.github/workflows/deploy.yml`) à chaque push sur la branche `main`.

## Stack technique

- React 18
- Vite
- Tailwind CSS 3
- Inter (Google Fonts)
- IntersectionObserver pour les animations au scroll
- SVG / CSS animations pour les pictogrammes et le mouvement
